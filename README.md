# analise_exportacao_agro
Este projeto desenvolvi com o objetivo de analisar dados de exportação e transformar essas informações em algo útil para apoiar decisões no setor do agronegócio.

A partir dos dados, foi possível entender como as exportações se comportam ao longo do tempo, quais estados se destacam e em quais períodos há maior volume de vendas. Neste caso, o foco foi nos estados da Bahia e Pernambuco, que estão entre os maiores exportadores de frutas do Brasil. 

Essas informações são importantes porque ajudam produtores e empresas a compreender melhor o mercado, permitindo um planejamento mais eficiente da produção e da logística.

A análise também contribui para identificar oportunidades de crescimento e ajustar estratégias comerciais com base em dados reais, reduzindo a dependência de decisões baseadas apenas em suposições. 

De forma geral, o projeto demonstra como o uso de dados pode trazer mais clareza, reduzir riscos e melhorar os resultados no agronegócio, apoiando decisões como o que produzir, quando vender e onde investir.
