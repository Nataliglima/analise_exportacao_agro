# 📊 Análise de Dados de Exportação no Agronegócio

# 📊 Agribusiness Export Data Analysis

---

## 🎯 Objetivo | Objective

**PT-BR:**
Este projeto foi desenvolvido com o objetivo de analisar dados de exportação e transformar essas informações em insights estratégicos para apoiar decisões no setor do agronegócio.

**EN:**
This project was developed to analyze export data and transform it into strategic insights to support decision-making in the agribusiness sector.

---

## 📊 Contexto da Análise | Analysis Context

**PT-BR:**
A análise foca nos estados da Bahia e Pernambuco, que estão entre os maiores exportadores de frutas do Brasil.

**EN:**
The analysis focuses on the states of Bahia and Pernambuco, which are among the largest fruit exporters in Brazil.

---

## 🔍 Principais Insights | Key Insights

* 📈 Crescimento das exportações ao longo do tempo
* 📅 Sazonalidade nas vendas
* 🌎 Estados com maior participação nas exportações
* 💰 Potencial de aumento de receita com simulação de crescimento

---

## 🛠️ Ferramentas Utilizadas | Tools Used

## ⚙️ Como Executar | How to Run

**PT-BR:**

1. Baixe o arquivo `dados_brutos_exportacao.xlsx`
2. Abra o notebook `analise_exportacao_agro.ipynb` no Google Colab ou Jupyter
3. Execute as células para realizar o tratamento e análise dos dados

**EN:**

1. Download the file `dados_brutos_exportacao.xlsx`
2. Open the notebook `analise_exportacao_agro.ipynb` using Google Colab or Jupyter
3. Run the cells to process and analyze the data

---

## 📁 Fonte dos Dados | Data Source

**PT-BR:**
Dados obtidos a partir do Comex Stat, sistema oficial de comércio exterior do Brasil.

**EN:**
Data obtained from Comex Stat, Brazil’s official foreign trade system.

**PT-BR:**
O projeto demonstra como a análise de dados pode apoiar decisões estratégicas no agronegócio, como planejamento de produção, definição de estratégias comerciais e identificação de oportunidades.

**EN:**
The project demonstrates how data analysis can support strategic decisions in agribusiness, such as production planning, commercial strategies, and identifying market opportunities.
